Xontrib Add Variable
===============================

version number: 0.1.0

author: [Erick Tucto]

Overview
--------

This package allow add new variable.

Installation / Usage
--------------------

To install use pip:

    $ xpip install xontrib-add-variable


Or clone the repo:

    $ git clone https://github.com/ErickTucto/xontrib-add-variable.git
    $ xpip install ./xontrib-add-variable

Add on `~/.xonshrc`

    xontrib load add_variable
