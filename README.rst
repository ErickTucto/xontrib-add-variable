Xontrib Add Variable
===============================

version number: 0.1.0

author: `Erick Tucto`_

Overview
--------

This package allow add new variable.

Installation / Usage
--------------------

To install use pip:

.. code:: bash
    pip3 install xontrib-add-variable


Add on `~/.xonshrc`

.. code:: python
    xontrib load add_variable
